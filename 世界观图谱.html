堕肉之歌

在万象尚未分名、群星尚未悬于高天之前，
最先自无垠中升起的，不是光，不是火，不是生命的第一声啼哭。
最先诞生于宇宙深处的，乃是——秩序。

秩序

起源
万物之始

它们无名，无相，无始，无终。它们更接近于法则本身——时间之所以流，空间之所以开，因果之所以相衔，万有之所以存在的根源显现。后来的众生不能理解这样的存在，便只能以最古老、也最接近敬畏的称呼，唤它们为——神明。

→ 显现为：神明

神明

起源
法则的显现

神明并非全知。它们知晓自己肩负职责，却不知由谁授予；握有伟力，却不知为何而生；各自趋向某一类秩序，却看不见秩序尽头的边界。

于是，在不可计数的岁月里，它们于沉默中彼此印证，于永恒中不断试验，探入法则更深之处，既观宇宙，也观自身。而每一次试验，都会留下痕迹。

它们造这一切，非出于慈悲，亦非为了统御，而是一场容纳无数世界、横跨无数纪元的实验——探究秩序如何诞生、延续与走向终末。

它们的真正职责，从来不是抽象地守护某种完美无瑕的秩序——而是阻止宇宙坠向崩坏。

规则复合体

起源
试验的残片

规则与规则相触，相嵌，相斥，相融；有些结合纯净而安定，有些结合却生出扭曲与错位。于是，无数不属于正常秩序之物诞生了。它们有的怪诞，有的危险，有的超出理性所能承受的边界。它们不是单一的法则，而是数重秩序在错误重叠之中孕出的异质残片。

这些复合体多半残缺、不稳，也不真正归属于任何一条纯粹的秩序之路。神明没有将它们留在正常宇宙之中，而是将其隔离、封缄、观测，如同将尚未熄灭的余烬置于不可触碰的圣匣之中。

规则复合体本身，便是秩序内部最不纯粹、最不稳固，也最接近失序之物。当外部混乱逼近时，最先发生共鸣的，恰恰是它们。

鱼缸

构造
实验之界

诸神以自身所处的真实宇宙为原型，筑起一座封闭的实验之界；在其中重新铺陈法则，播撒生命，推演文明，并将无数纪元、无数世界、无数可能，一并纳入这场漫长而宏大的求索之中。

于是，宏大的边界被建立——一重将世界与深渊、存续与崩坏彼此划开的界限。后世少数文明在支离破碎的神话中称呼它——鱼缸。

然而，这座鱼缸并非单层。它有两重天地。

← 创造者：神明 → 包含：内层 → 包含：夹层 → 包含：外层边界

内层

构造
万族栖居之地

最深之处，是内层。那是无数世界被安放之地。在那里，神明重新铺设宇宙的根基：令时间有河道可循，令空间有疆域可分，令因果彼此牵连，令存在获得重量与意义。

而在这共同的基石之上，不同区域、不同星辰、不同世界，又因底层参数的差异，而显现出截然不同的法则面貌。有的世界更易承载灵魂，有的世界更易唤起元素；有的偏向意志之显现，有的容许概念投下真实的影。于是，生命在其间诞生，文明在其间生长，群星之下，万族各循其道。

← 属于：鱼缸 → 孕育：内层文明 ← 渗透：污染

夹层

构造
封印之地 · 缓冲之域

在内层之外、外层边界之内，有一片浩瀚而凶险的夹层——诸神用以封印、容纳规则复合体的过渡领域。无数试验残片、畸变法则、未完成的构造与彼此冲突的神性余烬，皆被放逐于此。

夹层既是封印之地也是缓冲之地，将一切不容于正常秩序之物阻于内层之外，承担着实验宇宙最危险的一环。

对于内层众生而言，夹层既是禁忌，也是圣地。那里散落着远古遗迹、异常圣物、失落法则，以及通向更高真理的碎片。许多种族将步入夹层视为朝圣、试炼、飞升，或逐神之路；许多王朝与学派在那里寻找力量、答案、命运与超越。而这一切，并非僭越——这是神明所默许之事。

← 属于：鱼缸 → 封印：规则复合体 ← 朝圣试炼：内层文明 ← 入侵：混乱邪神

外层边界

构造
最后的屏障

外层边界是实验宇宙的最外围屏障，以法则本身铸就，将鱼缸内的秩序与外部混乱深渊隔绝。

若外层破碎，则夹层沦陷；若夹层沦陷，则内层诸界将被逐一开启。外层边界承受着无序最持久、最猛烈的撞击。

内层文明

文明
万族各循其道

纪元如潮，一重又一重地退去。内层诸界相继苏醒，文明于群星之间蔓延如林。不同的世界因底层参数的差异，孕育出截然不同的文明形态。有些文明走上了科技之道，有些文明走上了魔法之道。两者并非敌对，也并非彼此否定的真理——它们不过是生命理解世界、回应法则的两种方式。

许多文明仍以为灾祸只是某场天灾、神罚，或时代更迭的浩劫；却少有人知道，他们脚下的历史，头顶的群星，远方的诸界，与神话中不可言说的禁地，早已被卷入一场关乎宇宙存亡的战争之中。

← 孕育于：内层 → 发展路径：科技之道 → 发展路径：魔法之道 → 朝圣试炼：夹层

混乱宇宙

深渊
秩序的尸骸

在更遥远、更古老，也更可怖的层面上，还存在着另一类宇宙。它并非自始便与秩序对立，而是秩序宇宙崩坏之后留下的尸骸。

当一个宇宙无力维持自身法则，时间失去流向，空间失去界限，因果失去束缚——秩序在崩塌中腐烂。腐烂后所剩的，便是混乱宇宙。

混乱宇宙是正常世界死后仍在抽搐、膨胀、腐败的遗骸。没有稳定结构，没有可存留的规律。它无序地吞噬、同化周围一切，令万物失去差异，令所有道路被抹平成同一种混沌。

→ 孕生：混乱邪神

混乱邪神

深渊
崩坏法则的畸变意志

混乱宇宙的尸骸中会孕生更恐怖的存在——崩坏法则的残骸，失控规则在无尽扭结与腐败中形成的畸变意志。它们没有完整形体，不理解创造与共存，只知吞噬、同化与扩张。

当漫长岁月之外的混乱渐渐逼近这座实验宇宙时，最先发生共鸣的，并不是内层那些完整的世界，而恰恰是夹层中沉睡的规则复合体。它们开始苏醒，开始回应，开始成为裂缝——化作外部邪神探入鱼缸的触须、眼睛与门扉。

污染渗透

冲突
无法隔绝的侵蚀

污染终究无法被完全隔绝。当边界震荡，当夹层破损，当规则复合体被强行撬开，来自混乱的碎片、回响与污秽，便不可避免地渗入鱼缸之内，落进那些原本稳定运转的世界。

于是，有的星球诞生出无法解释的畸变生命；有的王朝在梦里听见不属于此界的低语；有的法师召来的不再只是元素与概念；有的学者在公式尽头窥见会反过来吞噬理性的深渊。

混乱并非一次性降临。它先是渗漏，继而扎根，最终才试图开启真正的大门。
← 引发：混乱邪神 → 渗透：内层

宇宙战争

冲突
秩序与混乱的终极对决

自宇宙初生以来最古老的战争，终于开始。一边，是被塑造、被维系、被无数文明栖居其上的秩序宇宙；另一边，是由崩坏世界堆叠而成的混乱深渊，以及自那些宇宙尸骸中孕出的邪神。

外层边界承受无序的撞击，夹层成为率先燃起战火的封印战场，内层诸界则在毫无察觉之时，迎来灾厄最初的影。

若实验中断，若边界崩裂，若内层与夹层一同滑入失序，那么神明所追寻的一切，都将与宇宙一道腐烂。
← 秩序一方：神明 ← 混乱一方：混乱邪神
